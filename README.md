# 🍍 엉클비 금융 계산기 (UncleB Finance Calculator)

예금·적금·연금·투자·대출을 한 곳에서 계산하는 웹 계산기입니다.
원본 엑셀(`파인애플 금융계산기.xlsx`)의 9개 시트 수식을 **Microsoft Excel 재무함수와 1원 단위까지 동일하게** 재현하고, **연금(노후 설계) 계산기**를 추가했습니다.

> 디자인 시스템: **Montage (Wanted Web Design System, WDS)** — [wanteddev/montage-web](https://github.com/wanteddev/montage-web)의 디자인 토큰(컬러·타이포·그림자) 적용.

## ✨ 기능 (5개 탭)

| 탭 | 내용 |
|---|---|
| **예금 계산기** | 거치식 — 월복리/연단리 **버튼 토글**, 세금 3구간 실수령액 + 물가 반영 현재가치 |
| **적금 계산기** | 적립식 — 월복리/연단리 **버튼 토글**, 세금 3구간 실수령액 + 물가 반영 현재가치 |
| **연금 계산기** ⭐ | 2단계 노후 시뮬레이터: ① 적립(축적, FV) → ② 수령(인출, PMT). 원금소진형/이자수령형, 물가 반영 실질 연금 |
| **투자 계산기** | 빠른계산(미래가치·현재가치·목돈모으기) · 월지급식펀드 · 투자자산배분 · 거치+적립 투자 |
| **대출 계산기** | 원리금균등 · 원금균등 · 만기일시 상환 비교 (회차별 상환 스케줄) |

## 🧮 구현한 엑셀 재무함수

`FV` · `PV` · `PMT` · `IPMT` · `PPMT` 를 엑셀과 동일한 부호·정밀도로 구현 ([js/fin.js](js/fin.js)).

## 📂 구조

```
pineapple-finance/
├── index.html            # 화면 구조 (예금/적금/연금/투자/대출)
├── vercel.json           # Vercel 배포 설정
├── css/styles.css        # Montage(WDS) 디자인 토큰 기반 UI
└── js/
    ├── fin.js            # 엑셀 재무함수 + 포맷 유틸
    ├── calculators.js    # 시트별 계산 로직 + 연금 시뮬레이터 (셀 참조 주석 포함)
    ├── charts.js         # 순수 SVG 차트 (의존성 0)
    └── app.js            # 입력↔계산↔표/차트 실시간 연동
```

## ⚠️ 원본 엑셀 대비 보정 사항

- **투자자산배분 단리 버킷**: 원본의 253~264개월 구간 이자 가산 오류를 수학적으로 올바른 **12개월 주기**로 정정.
- **대출 거치기간**: 원본은 미구현(메모 존재) → 표준 거치(이자만 납부 후 분할상환) 옵션 추가.
