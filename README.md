# 🍍 파인애플 금융계산기 (Pineapple Finance Calculator)

예금·적금·투자·연금·대출을 한 곳에서 계산하는 **완전 독립형 웹 계산기**입니다.
원본 엑셀(`파인애플 금융계산기.xlsx`)의 9개 시트 수식을 **Microsoft Excel 재무함수와 1원 단위까지 동일하게** 재현했습니다.

> 외부 라이브러리·CDN 의존성이 전혀 없어, `index.html`을 더블클릭하면 오프라인에서도 바로 동작합니다.

## ✨ 기능 (5개 탭)

| 탭 | 내용 |
|---|---|
| **종합 계산기** | 미래가치 · 현재가치 · 목돈모으기 · 예금/적금(월복리·연단리) — 세금 3구간 실수령액 + 물가 반영 현재가치 |
| **월지급식 펀드** | 분배금 재투자 모델 (보유좌수 = 원금 ÷ 기준가 × 1,000) |
| **투자자산 배분** | 금리형(단리)·채권형(월복리)·주식형(월복리) 비율 분산 적립 |
| **거치 + 적립 투자** | 초기 목돈(거치식) + 매월 적립식 혼합 전략 예상액 |
| **대출 계산기** | 원리금균등 · 원금균등 · 만기일시 상환 비교 (회차별 상환 스케줄) |

## 🧮 구현한 엑셀 재무함수

`FV` · `PV` · `PMT` · `IPMT` · `PPMT` 를 엑셀과 동일한 부호·정밀도로 구현 ([js/fin.js](js/fin.js)).

## 🚀 실행

```bash
# 방법 1: 그냥 index.html 더블클릭

# 방법 2: 로컬 서버
python -m http.server 8753
# → http://localhost:8753
```

## 📂 구조

```
pineapple-finance/
├── index.html            # 화면 구조
├── css/styles.css        # 파인애플 테마 UI
└── js/
    ├── fin.js            # 엑셀 재무함수 + 포맷 유틸
    ├── calculators.js    # 9개 시트 계산 로직 (셀 참조 주석 포함)
    ├── charts.js         # 순수 SVG 차트 (의존성 0)
    └── app.js            # 입력↔계산↔표/차트 실시간 연동
```

## ⚠️ 원본 엑셀 대비 보정 사항

- **투자자산배분 단리 버킷**: 원본의 253~264개월 구간 이자 가산 오류를 수학적으로 올바른 **12개월 주기**로 정정.
- **대출 거치기간**: 원본은 미구현(메모 존재) → 표준 거치(이자만 납부 후 분할상환) 옵션 추가.
